# 油猴脚本快速下载指南

## 🎯 为什么选择油猴脚本？

### ⚡ 速度优势：
- **Playwright 自动化**：30-60 分钟，需要脚本运行
- **油猴脚本**：5-10 分钟，只需点击转存！

---

## 📦 安装步骤（2 分钟）

### 1️⃣ 安装 Tampermonkey 扩展
1. 已自动打开 Chrome 扩展商店
2. 点击"添加至 Chrome"
3. 等待安装完成

### 2️⃣ 安装网盘脚本
- 已自动打开脚本页面
- 点击右上角"安装此脚本"
- 选择"添加到 Tampermonkey"
- 等待 30 秒

---

## 🚀 使用流程

### 方法 A：批量转存到网盘（推荐）

```bash
cd "/Users/chi/Scripts/02-多媒体处理/MFCPX-Downloader"
./batch_open_for_tampermonkey.sh
```

**油猴脚本会自动在每个页面上显示：**
- 📥 转存按钮
- 📋 下载直链

**你只需要：**
1. 每批打开 5 个链接（自动）
2. 在每个页面点击"转存到我的网盘"
3. 选择保存位置（第一次）：`/FCPX插件`
4. 全部转存完成后运行下载脚本

**时间：** 10-15 分钟

---

### 方法 B：复制直链批量下载

1. 打开 5 个链接（自动）
2. 油猴脚本显示下载直链
3. 复制所有直链到文本文件
4. 用 aria2c 批量下载

```bash
aria2c --input-file=links.txt --dir=/Volumes/BigDisk/Download/FCPX-Plugins
```

**时间：** 10-15 分钟

---

## ✅ 安装好 Tampermonkey 和脚本后
