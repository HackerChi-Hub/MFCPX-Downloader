# 百度网盘客户端批量下载 FCPX 插件

## 📥 第一步：安装百度网盘客户端

如果没有安装，下载 Mac 版：
- 官网：https://pan.baidu.com/download
- 或在 App Store 搜索"百度网盘"

## 📋 第二步：批量导入链接

### 方法 A：使用导入文件（推荐）

1. **打开导入文件**
   ```bash
   open "/Users/chi/Scripts/02-多媒体处理/MFCPX-Downloader/baidu_links_import.txt"
   ```

2. **在百度网盘客户端中**
   - 点击左侧"分享"或"链接"选项卡
   - 找到"导入链接"或"批量导入"按钮
   - 选择 `baidu_links_import.txt` 文件
   - 点击"导入"

### 方法 B：手动复制粘贴

1. 打开 `baidu_urls_only.txt`
2. 全部复制（Cmd+A, Cmd+C）
3. 在百度网盘客户端中找到"导入链接"
4. 粘贴链接

## ⚙️ 第三步：批量保存到网盘

导入后，客户端会显示所有链接：
- 自动识别密码（已在文件中）
- 如有验证码，输入一次即可
- 全选所有链接（Cmd+A）
- 点击"保存到我的网盘"
- 选择或创建文件夹：`/FCPX插件`
- 等待全部保存完成

## 📥 第四步：下载到本地

### 方案 A：用客户端直接下载
- 在客户端找到 `我的网盘 > FCP插件` 文件夹
- 全选所有文件
- 点击"下载"
- 选择保存位置：`/Volumes/BigDisk/Download/FCPX-Plugins/`

### 方案 B：用 Alist 批量下载（更快，推荐）
客户端保存完成后，运行：
```bash
cd "/Users/chi/Scripts/02-多媒体处理/MFCPX-Downloader"
python3 baidu_batch_downloader.py download
```

## ✅ 完成

## ❓ 常见问题

**Q: 链接导入失败？**
A: 尝试分批导入，每次 10-20 个链接

**Q: 密码错误？**
A: 检查 `baidu_links_import.txt` 中的密码是否正确

**Q: 下载很慢？**
A: 使用方案 B（Alist 下载）可以充分利用带宽
