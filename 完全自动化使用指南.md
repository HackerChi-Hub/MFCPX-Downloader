# FCPX 插件完全自动化下载指南

## 🎯 方案：使用 Playwright 自动保存

### ✨ 特点：
- ✅ **全自动**：自动打开链接、输入密码、点击保存
- ✅ **批量处理**：81 个链接自动分批处理
- ✅ **断点续传**：支持失败重试
- ⚠️ **验证码**：如遇到验证码，会暂停等你手动输入（仅需一次）

---

## 📋 使用步骤

### 第 1 步：启动自动保存

```bash
cd "/Users/chi/Scripts/02-多媒体处理/MFCPX-Downloader"
python3 auto_save_playwright.py
```

### 第 2 步：等待完成

脚本会自动：
1. 打开你的 Chrome 浏览器（使用 Profile 7）
2. 逐个访问 81 个分享链接
3. 自动输入密码
4. 自动点击"保存到我的网盘"
5. **如遇验证码**：暂停并在终端提示你输入

**预计时间：** 30-60 分钟

### 第 3 步：批量下载到本地

保存完成后，运行：

```bash
python3 baidu_batch_downloader.py download
```

**预计时间：** 取决于网速（约 5-10GB）

## ⚠️ 注意事项

### 1. 浏览器登录状态
- 确保你的 Chrome Profile 7 已登录百度网盘
- 如未登录，脚本启动时会打开浏览器让你登录

### 2. 验证码处理
- **大部分链接不需要验证码**
- 如遇到，脚本会暂停并在终端提示
- 输入验证码后按 Enter 继续

### 3. 保存位置
- 默认保存到网盘根目录或上次使用的位置
- 如需指定位置，可以手动在第一个链接时选择

### 4. 网络连接
- 保持稳定的网络连接
- 如中断，可以重新运行（支持断点续传）

## 🆘 故障排除

### 问题 1：脚本卡住不动
```bash
# 停止脚本（Ctrl+C）
# 重新运行即可（会跳过已保存的）
python3 auto_save_playwright.py
```

### 问题 2：浏览器未登录百度网盘
- 脚本会自动打开浏览器
- 手动登录百度网盘
- 刷新页面后脚本会继续

### 问题 3：验证码一直出现
- 这是百度网盘的安全机制
- 建议分多次运行，每次处理 10-20 个链接
- 修改脚本中的 `batch_size = 5` 为 `batch_size = 3`
